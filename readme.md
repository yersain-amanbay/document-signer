### Installation:

```bash
git clone "https://github.com/yersain-amanbay/document-signer.git"
cd document-signer
pip install -r requirements.txt
touch .env
```

### Env File Format:

```bash
SIGN="./sign/sign.png"
SIGN_WIDTH=1200
SIGN_HEIGHT=2000
INPUT="./input/"
OUTPUT="./output/"
TMP_FOLDER="./tmp_images/"
```
